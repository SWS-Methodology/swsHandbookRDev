PoU in the SWS

Chapter 1 PoU Framework

The Prevalence of Undernourishment (PoU) is such a complex statistical process ran by the Food Security and Nutrition Statistics (FSNS) team in ESS.

Its workflow is composed by several SWS objects - four plugins, three data sets and nine data tables. The image below illustrates those objects and how they interact with each other to compute the SDG indicator 2.1.1 in the system.

It is worth highlighting that this is a proposal and changes may apply.

1.1 Data Migration

During the implementation of any statistical output in the SWS, one of the key moments is the migration of the legacy data. Seen for non-SWS users as only one simple operation, this step may become a challenge if the previous working system does not have a well-structured database. This is because the metadata attached to the data and its history itself is as important as the final output and there were cases in previous projects that the SWS/IT team had to deal with messy databases.

Another point is that, there might have cases where areas/items/elements conversions need to be applied. Not to mention the conversion of flags to the SWS - SDMX flag system.

For each disseminated data to be migrated to SWS there is a need to convert:

  • areas to geographicAreaM49;
  • items to measuredItemCPC (or another code list that contains those items);
  • elements to measuredElement (or another code list that contains those elements);
  • flags to SWS flag system (flagObservationStatus and flagMethod).

In this specific project, there is a need to create additional codes (in the current code lists) and also new code lists to accommodate all the input and output data.

1.2 Input Data

FAO is responsible for the dissemination of several statistical outputs including data on agriculture production and trade, forestry, fisheries, fertilizers, pesticides, producer prices and others. Along with the data, it is also important for transparency purposes to release information regarding its quality. Thus this book describes in details the process to compile the QI for each of these outputs, as they are composed of differents kinds of elements, items and areas. Moreover, each statistical process may have its own data processing, from harvesting to storage, with peculiarities that need to be taken into consideration when calculating the QI.

1.2.1 Dietary Energy Supply

Dietary Energy Supply (DES) in kilocalories per person per day in a given country is “the most important output of the Food Balance Sheet (FBS)”, according to FAOSTAT. Its correspondent data set in the SWS is in the domain Disseminated Datasets, data set FBS balanced diss, from year 2010. Its legacy data with information before 2010 using the old methodology can also be found in the SWS.

1.2.2 Population by age groups

1.2.3 Crude Birth Rate

Crude Birth Rate is the annual number of live births per 1,000 population. Its source are both the civil registration and surveys/censuses. Its correspondent data set in the SWS is in the domain Population, data set Population.

1.2.4 Under-Five Mortality Rate

Under-Five Mortality Rate are estimates developed by the UN Inter-agency Group for Child Mortality Estimation ( UNICEF, WHO, World Bank, UN DESA Population Division ) at childmortality.org. Its correspondent data set in the SWS is in the domain PoU, data set …

1.2.5 Height Distribution by age-sex classes

Height Distribution by age-sex classes is collected by the FSNS team from various sources. These data are stored as a Datatable object in the SWS, domain PoU.

1.2.6 Weight Distribution for age/height/length

Weight Distribution for age/height/length are a set of z-scores tables disseminated by the World Health Organization (WHO). Their correspondent data are found in SWS/Datatables, in the domain PoU.

1.2.7 Weight gain/Energy Requirement per KG of Weight Gain

Weight gain/Energy Requirement per KG of Weight Gain data by gender and age class are taken from “WHO, Measuring change in nutritional status - Guidelines for assessing the nutritional impact of supplementary feeding programmes for vulnerable people, World Health Organization, 1983, Geneva”, tables 22, 23, 24, and 25. For MDER, ADER, and XDER we use respectively the 5th, the 50th and the 95th percentiles. Their correspondent data are found in SWS/Datatables, in the domain PoU.

1.2.8 XCBS

XCBS is the percentage of changes in food availability. It comes from an unique database created and continuously kept up to date by GIEWS/Basic Foodstuffs Team of the Markets and Trade Division of FAO. The data are shared to the FSNS team through an excel file. Their correspondent data are found in SWS/Datatables, in the domain PoU.

1.2.9 Pregnancy Allowance (PAL)

[Pregnancy Allowance (PAL)] is one parameter with value 210 and is in the SWS/Datatables, in the domain PoU.
